# TEDRADIO

![demo](https://raw.githubusercontent.com/wiki/toywonder/tedradio/img/tedradio_demo_full_2.gif)

## Description
TEDRadioは、外国語のTEDでも日本語音声で聞けるTED日本語音声メディアです。字幕を見るために目が離せなくなっていたTEDも、楽に聞けるようになりました。有名TEDを是非日本語で聞いてみてください！
